End points of interest

  • SpringBoot Actuator
    Spring boot actuator exposes a number of useful links. Links such as /beans that shows all classes tracked by spring, /health which shows application status, /info which has been customized to show most pertinent info

Swagger/Open API

Swagger, now being rebranded as OpenAPI, exposes useful Rest info and a means to test things out.
  • API Docs
  • UI Console

H2 database (if enabled)

H2 is an embedded database backed by either memory or file storage. It is useful for testing/development since you don't have to install and configure an actual database. This console allows for you to interact with the database.
  • H2 Console

Sample Data

Self explanatory I hope, these options can be used to load sample data, import json, or export json from the database. After you load/import the database you can use either the H2 link to query and see what was loaded or use the Swagger links to retrieve the json for the loaded data.
  • Load Sample Data
  • Export
  • Import wasn't implemented but possible if you wish to implement
